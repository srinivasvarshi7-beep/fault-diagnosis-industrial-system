# AI-Based Fault Diagnosis System

## Project Description
This project is an AI-based Fault Diagnosis System developed using Python and Streamlit.

The project demonstrates:
- Rule-Based Systems
- Search Techniques
- Constraint Satisfaction
- Probabilistic Reasoning
- Machine Learning
- Intelligent Agents

## Technologies Used
- Python
- Streamlit
- Scikit-learn
- Pandas

## Deployment
This project is deployed using Streamlit Cloud.

## Run Locally
Install dependencies:

pip install -r requirements.txt

Run the application:

streamlit run app.py
